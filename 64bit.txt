Hello Linux user.

Looking to install Mozilla Firefox?

Mozilla Firefox comes standard in 64-bit. 
This release folder contains the necessary scripts to install those releases

If you are you require 32-bit editions, please look within the 32bit folder.

Installation instructions can be found within how-to-install.txt

For normal end-user usage, you need only pick between 32-bit or 64-bit (not both).	 
  
-- Happy Browsing!
